### Controls
    
    Key           | Action
    --------------|-------------------------
    <kbd>Up</kbd> | Slide the board upwards
    <kbd>Down</kbd>| Slide the board downwards
    <kbd>Left
    
    
- Keyboard arrows for gameplay
- 'm' to change the color scheme
- 'q' to quit the game
